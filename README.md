# Power BI Sales Analysis Report

This project presents an **interactive Power BI dashboard** built using an Amazon sales dataset to help understand sales performance in a simple and visual way.

## Overview
The report is designed so that **anyone can easily interpret the data** without technical knowledge. It focuses on identifying sales patterns and quantity trends.

## Report Pages
- **Quantity-wise Analysis**  
  Insights into product quantities sold across different categories.
  
- **Sales-wise Analysis**  
  Analysis of revenue performance to understand sales contribution and trends.

## Tools Used
- Power BI
- Data Cleaning & Analysis

## Purpose
To demonstrate **data visualization, business insight generation, and dashboard design** using Power BI.
